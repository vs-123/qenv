* QENV

*qenv* is a simple, POSIX-compliant, zero-dependency environment variable inspection utility tool designed for quick and memory-efficient filtering across diverse shell environments.

** FEATURES

- **[STREAM-ORIENTED TRAVERSAL]** -- Direct iteration through the global environment pointer array. This ensures constant memory usage regardless of the environment's size. This satisfies the requirement to handle multi-megabyte environment blocks without buffering the entire output set.
  
- **[PORTABLE MEMORY CALCULATION]** -- Flag =-v= is available to provide a total environment block size measurement by aggregating the length of all environment strings and the pointer array overhead. This maintainins POSIX portability by avoiding non-standard filesystem interfaces.
